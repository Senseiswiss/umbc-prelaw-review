UMBC Pre-Law Review — PDF Storage
──────────────────────────────────
Drop published article PDFs into this folder.

Naming convention:
  vol{V}-iss{I}-{lastname}-{shorttitle}.pdf

Examples:
  vol1-iss1-chen-fourth-amendment.pdf
  vol1-iss1-rodriguez-substantive-due-process.pdf

Then reference the file in data/articles.js as:
  pdfPath: "pdfs/vol1-iss1-chen-fourth-amendment.pdf"
